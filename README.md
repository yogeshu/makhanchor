# makhanchor

A modern, interactive author portfolio and landing page built with Next.js. This is a professional showcase for an author featuring book promotions, merchandise sales, and social media integration.

## Features

- **Author Portfolio**: Beautiful author page with profile and background information
- **Book Promotions**: Showcase books with promotional pricing and coupon code management
- **E-commerce Integration**: Buy buttons with pricing, discounts, and coupon codes
- **Social Media Links**: Direct integration with Instagram, YouTube, and other platforms
- **Dark Mode Support**: Responsive design with dark mode compatibility
- **Analytics Tracking**: Google Analytics integration for tracking user interactions
- **Responsive Design**: Mobile-friendly layout optimized for all screen sizes
- **Performance Optimized**: Built with Next.js 16 and React 19 with React Compiler enabled

## Tech Stack

- **Framework**: Next.js 16.1.1
- **UI Library**: React 19.2.3
- **Styling**: Tailwind CSS 3.4.1
- **Icons**: Lucide React
- **Analytics**: Google Analytics
- **Code Quality**: ESLint, Babel Plugin React Compiler
